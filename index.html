Chapter 1:

Getting started with HTML:

HTML
(Hypertext Markup Language) uses a markup system composed of elements which represent specific content. Markup means that with HTML you declare what is presented to a viewer, not how it is presented.
HTML is sometimes called a programming language but it has no logic, so is a
markup language
. HTML tags provide semantic meaning and machine-readability to the content in the page
An element usually consists of an opening tag (<element_name>), a closing tag (</element_name>), which contain the element's name surrounded by angle brackets, and the content in between: <element_name>...content...</element_name>
HTML - index.html
                
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Hello!</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <p>This is a simple paragraph.</p>
    </body>
</html>
Chapter 2:

Headings:

HTML provides not only plain paragraph tags, but six separate header tags to indicate headings of various sizes and thicknesses. Enumerated as
heading 1
through
heading 6
, heading 1 has the largest and thickest text while heading 6 is the smallest and thinnest, down to the paragraph level. This topic details proper usage of these tags.
HTML - index.html
                
<h1>Heading 1 </h1>    <!-- h1 is the largest heading -->
<h2>Heading 2 </h2>
<h3>Heading 3 </h3>
<h4>Heading 4 </h4>
<h5>Heading 5 </h5>
<h6>Heading 6 </h6>    <!-- h6 is the smallest heading -->

Chapter 4:

Text Formatting:

While most HTML tags are used to create elements, HTML also provides in-text formatting tags to apply specific text-related styles to portions of text. This topic includes examples of HTML text formatting such as highlighting, bolding, underlining, subscript, and stricken text.

Section 4.1: Highlighting:

The
<mark> </mark>
element is new in HTML5 and is used to mark or highlight text in a document
HTML - index.html
                    
<p>Hello, I Love <mark>HTML</mark> Very Much.</p>

Section 4.2: Bold, Italic, and Underline:

The
<strong> </strong>
and
<b> </b>
tags are used for transform the text to Bold style
<strong>
is used to indicate that the text is fundamentally or semantically important to the surrounding text, while
<b>
indicates no such importance and simply represents text that should be bolded.
HTML - index.html
                    
<strong>Bold Text Here </strong>    <!-- Important Text -->
<b>Bold Text Here </b>              <!-- Normal Bold Text  -->
The
<em> </em>
and
<i> </i>
tags are used for transform the text to italic style
<em>
is used to indicate that the text should have extra emphasis that should be stressed, while
<i>
simply represents text which should be set off from the normal text around it.
HTML - index.html
                    
<em>Italicized Text Here </em>
<i>Italicized Text Here </i>
While the
<u> </u>
element itself was deprecated in HTMl 4, it was reintroduced with alternate semantic meaning in HTML 5 - to represent an unarticulated, non-textual annotation.
  You might use such a rendering to indicate misspelled text on the page, or for a Chinese proper name mark.
HTML - index.html
                        
<p>This paragraph contains some <u>mispelled</u> text.</p>

Section 4.3: Abbreviation:

To mark some expression as an abbreviation, use
<abbr> </abbr>
tag.
If present, the title attribute is used to present the full description of such abbreviation.
HTML - index.html
                    
<p>I like to write <abbr title="Hypertext Markup Language">HTML</abbr>!</p>

Section 4.4: Inserted, Deleted, or Stricken:

To mark text as inserted, use the
<ins> </ins>
tag, to mark text as deleted, use the
<del> </del>
tag, and to strike through text, use the
<s> </s>
tag.
HTML - index.html
        
<ins> Add a new text </ins> <-- To mark text as inserted -->
<del> Delete a  text </del> <-- To mark text as deleted -->
<s> Text example here </s>  <-- To strike through text -->

Section 4.5: Superscript and Subscript:

To offset text either upward or downward you can use the tags
<sup> </sup>
and
<sub></sub>
HTML - index.html
        
<p> In Physics we use O<sup>2</sup> </p>
<p> In Maths we use 5<sub>5</sub> </p>
Chapter 5:

Anchors:

Anchor tags are commonly used to link separate webpages, but they can also be used to link between different places in a single document, often within table of contents or even launch external applications. This topic explains the implementation and application of HTML anchor tags in various roles.
To use the anchor tag in our page we use the
<a> </a>
tag, and we insert to it, the href , hreflang and target attributes.

Anchor Attributes

Attribute Name Attribute Role
href Specifies the destination address. It can be an absolute or relative URL, or the name of an anchor. An absolute URL is the complete URL of a website
hreflang Specifies the language of the resource linked by the href attribute
target pecifies where to open the link, e.g. in a new tab or window. Possible values are _blank, _self, _parent, _top.
href Specifies the destination address. It can be an absolute or relative URL, or the name of an anchor. An absolute URL is the complete URL of a website

Warning:

The href attribute in the anchor tag is mandatory, but the hreflang and target... are optionals
HTML - index.html
        
<a href="https://www.google.com" target="_blank"> Google </a>
<a href="#contact-section"> Contact-Us </a>
<a href="javascript:myFunction();"> Run The Function </a>
<a href="mailto:example@example.com?subject=test&body=message"> Email </a>
Chapter 6:

Lists:

HTML offers three ways for specifying lists: ordered lists, unordered lists, and description lists. Ordered lists use ordinal sequences to indicate the order of list elements, unordered lists use a defined symbol such as a bullet to list elements in no designated order, and description lists use indents to list elements with their children.

Section 6.1: Ordered lists:

An ordered list can be created with the
<ol> </ol>
tag and each list item can be created with the
<li> </li>
tag as in the example below:

Ordered list attributes

Attribute Name Attribute Role
start The list will start at this defined number, and continue incrementing by one as usual.
value by using the value attribute directly on a list item, you can override an ordered list's existing numbering system by restarting the numbering at a lower value.
reversed Reverse numbering is helpful if you're continually adding to a list, such as with new podcast episodes or presentations, and you want the most recent items to appear first.
type change the type of numeral shown in the list item marker by using the type attribute
HTML - index.html
        
<ol start="10">
    <li> python </li>
    <li> java </li>
    <li start="25"> kotlin </li>
    <li> C# </li>
</ol>

Section 6.2: Unordered lists:

An unordered list can be created with the
<ul> </ul>
tag and each list item can be created with the
<li> </li>
tag as shown by the example below:
HTML - index.html
        
<ul>
    <li> python </li>
    <li> java </li>
    <li> kotlin </li>
    <li> C# </li>
</ul>

Section 6.3: Description lists:

A description list can be created with the
<dl> </dl>
element. It consists of name-value groups, where the name is given in the
<dt> </dt>
element, and the value is given in the
<dd> </dd>
element.
HTML - index.html
        
<dl>
    <dt>term 1<dt>
    <dt>term 2<dt>
    <dd>Description of term 1 and term 2<dd>

    <dt>term 3<dt>
    <dd>Description of term 3<dd>
    <dd>Description of term 3<dd>
</dl>

Important note:

This type of lists may not be used for much, but it is a good idea to learn it out of knowledge.
Chapter 7:

Images and Media:

Section 7.1: Images:

To add an image to a page, use the image tag
<img />
.

Image attributes

Attribute Name Attribute Role
src Specifies the URL of the image.
srcset Images to use in different situations.
sizes Image sizes between breakpoints.
usemap Name of image map to use
alt Alternative text that should be displayed if for some reason the image could not be displayed.
width Specifies the width of the image
height Specifies the height of the image
HTML - index.html
        
<img src="../imgs/picture1.jpg" alt="Not Found Image" />

Section 7.2: Audio:

HTML5 provides a new standard for embedding an audio file on a web page.
You can embed an audio file to a page using the
<audio> </audio>
element.

Media attributes

Attribute Name Attribute Role
<source> Defines resources of the audio or video files
track Defines the text track for media elements.
controls Defines the text track for media elements.
autoplay Automatically start playing the media.
loop Automatically start playing the media
width Specifies the width of the image
height Specifies the height of the image
muted Plays the media without sound.
poster Assigns an image to display until a video is loaded.
HTML - index.html
        
<audio controls loop muted >
    <source src="../media/sound1.mp3" type="audio/mp3"> </source>
</audio>

Section 7.3: Video:

You can embed also a video to a webpage using the
<video> </video>
element:
HTML - index.html
        
<video controls loop muted poster="../imgs/load.jpg" >
    <source src="../media/video1.mp4" type="video/mp4"> </source>
</video>
